# strangeloop

Networked instrument looper pedal enabling remote collaboration between musicians. Built using **Flask**, **Raspberry Pi** & **ReactJS**, on an [ElectroSmash Pedal-Pi](https://www.electrosmash.com/pedal-pi) kit.
